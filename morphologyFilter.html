Instruction Header

This is a minimal cross platform version of my C#/.Net app: plantid

Select morphology characteristics from the drop down boxes.

Toggle the "On/Off" switches to query plants whose leaves have that characteristic. Only plants which contain "On" switch characteristics will be listed. If even one extra characteristic is not present, or match with the combination of other characteristics, the search results will yield zero.

Though properly identifying a plant requires a lot more than just knowing its leaf morphology, one can reliably deduce which plant species it likely belongs to.

Import an appropriate Json file

After selecting a file, click "Load" to confirm your selection.
